Fashion MNIST Image Classification using CNN

This project implements a Convolutional Neural Network (CNN) using TensorFlow and Keras to classify Fashion MNIST images into 10 clothing categories. It includes data preprocessing, model training, evaluation, prediction, and visualization of training accuracy and loss.

Features

- CNN-based image classification
- Image preprocessing and normalization
- Batch Normalization and Dropout
- Early Stopping to reduce overfitting
- Accuracy and loss visualization
- Prediction on test images

Tech Stack

- Python
- TensorFlow / Keras
- Matplotlib

Dataset

- Fashion MNIST (60,000 training images and 10,000 test images)
- 10 clothing categories
- 28×28 grayscale images

Results

- Test Accuracy: 91%
